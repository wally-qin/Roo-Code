## 🔴 **关键问题: Tree-sitter查询差异**

### 问题严重性
- **标识符提取缺失**: 无法准确获取函数/类名称
- **语言特性覆盖不足**: 缺失装饰器、lambda、生成器等
- **代码分块质量差**: 粗糙的结构分块vs精确的语义分块
- **搜索精确度低**: 无法基于函数名进行精确搜索

### 具体差异示例
```typescript
// TypeScript版本 - 完整查询 (74行)

    name: (identifier) @name.definition.class)) @definition.class
```
